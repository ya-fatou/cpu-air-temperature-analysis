The repository includes:
- Python script for data processing
- Sample CSV dataset
- PDF reports showcasing statistical analysis

 Project 1: Python Script
 File: tempLog.py  
- Python script that processes CPU and air temperature data from tempLog.csv file  
- Calculates median, mode, mean, minimum, and maximum values for CPU and air temperatures.  
- Includes verification of results using the pandas library.

Project 2: Research Methods Report
File: research-methods-report.pdf
- PDF report from the Research Methods module.
- Demonstrates statistical analysis using: Pearson correlation,Chi-square tests, One-way ANOVA
- Includes data exploration, hypothesis testing, results, and conclusions.
